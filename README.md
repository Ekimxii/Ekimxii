# Erin Kim

Data Science + Cognitive Science @ UC Berkeley

I build systems that make unstructured information interpretable across different areas of expertise. Most of my work lives at the intersection of NLP, retrieval systems, and figuring out when AI outputs are trustworthy enough to act on.

## Projects

**[CourseWise AI Chatbot](https://github.com/Ekimxii/CourseWise-Public)** : RAG-based chatbot for course transfer planning, built for registrars and advisors across 70+ institutions. Focused on retrieval architecture, chunking decisions, and evaluation frameworks for LLM output quality.

**[Echo Check](https://github.com/Ekimxii/Echo-Check)** : A diagnostic tool that tests whether AI systems validate emotional dependency on themselves. Three system prompt conditions, five scenarios, scored sycophancy output. Built with Flask and Claude Code.
